APPROACH SUMMARY: Convert the 2 dimensional array 'values' to map. Map key is the combination of first N-1 (N: array length) elements of each element array of 'values'. The last elements are the map values. By this way we don't need to search through the arrays, and the 'not ordered' is not an issue any more. The time complexity is O(N). The space complexity (created map in memory) is also O(N). For extremely large Json data, back end should provide pagination and front end should have 'Next' and 'Previous' buttons.
